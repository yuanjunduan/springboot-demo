# 精准运营平台

## 精准运营平台技术栈

主要使用到的技术有以下：

+ springboot 
+ springboot security
+ springboot jpa
+ swagger
+ joda time
+ fasterxml jackson
+ p6spy

项目采用前后分离框架，后端只需要系统 RESTful 风格的接口即可。

### others

+ 统一使用google code style作为我们的编码规范
+ 通过gitlab实现自动化的ci流程
